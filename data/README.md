# Датасеты, используемые в проекте

Датасеты в репозиторий **не добавляются**, чтобы не раздувать размер и не нарушать условия распространения.
Ниже указаны источники:

1. **Задача 1 – обнаружение фейковых новостей**

   - Датасет: Fake and real news dataset
   - Источник: Kaggle или указанный в задании набор (ссылка из методички)
   - Описание: <https://storage.yandexcloud.net/academy.ai/practica/fake_news.csv>
   - В репозитории используется файл `fake_news.csv`.

2. **Задача 2 – болезнь Паркинсона**

   - Датасет: UCI ML Parkinsons
   - Описание: <https://storage.yandexcloud.net/academy.ai/practica/parkinsons.names>
   - В репозитории используется файл `parkinsons.names`.

3. **Задача 3 – прогноз сдачи экзамена**

   - Датасет: Student Performance Data Set
   - Описание: <https://archive.ics.uci.edu/ml/datasets/student+performance>
   - В репозитории используется файл `student-mat.csv`.
